# Omniauth::HaveAPI

OmniAuth strategy using any HaveAPI based API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-haveapi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-haveapi
